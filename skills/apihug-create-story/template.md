- NEVER: Security annotations (@PreAuthorize, @Secured) or permission checks in code
- NEVER: `builder.done()` calls (controller handles it automatically)
- NEVER: RuntimeException for business errors (use `HopeErrorDetailException`)

### Source Set Constraints
- NEVER: Modify src/generated/ (READ-ONLY - Wire generated)
- NEVER: Create ServiceImpl manually (run `gradlew :module:wire`)
